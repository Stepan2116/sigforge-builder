### GitHub Sponsors

- **URL:** https://github.com/sponsors
- **Pool:** N/A (recurring monthly contributions)

- **Fit:** Open-source software, low overhead
- **Ask:** $5/$25/$100/$500 monthly tiers
- **Action:**

## 🟠 Medium effort — 2-4 hours each

### Anthropic Builders Programs / OpenAI dev grants

- **URL:** https://anthropic.com (check Builders portal); openai.com (varies)
- **Pool:** $5K-$50K per grant, irregular
- **Fit:** if we frame backtester or watchdog as AI-augmented
  (e.g., "use Claude to auto-generate strategy hypotheses from trade logs")
- **Ask:** $5K-$15K
- **Adapt cost:** new section in pitch about AI-augmentation
- **Note:** Lower fit unless we genuinely add an LLM layer to the framework.
  Don't fake-frame — reviewers see through it.

### Polygon Village Grants

- **Pool:** $1M+ ecosystem fund
- **Fit:** SigForge runs on Polygon today (Polymarket is on Polygon)
- **Ask:** $5K-$25K
- **Adapt cost:** emphasize Polygon-native strategy, on-chain attribution
- **Round timing:** continuous applications

### Arbitrum Foundation Grants

- **URL:** https://arbitrum.foundation
- **Pool:** $50M+ ecosystem fund
- **Fit:** if we build an Arbitrum prediction-market adapter (currently
  not in scope but possible day-60 milestone)
- **Ask:** depends on adapter scope, $10K-$50K
- **Action:** Lower priority unless Arbitrum prediction-market support
  becomes interesting

## Anti-pattern: don't spam

Apply to grants where the fit is real. Don't apply to every program
that exists. Reviewer cross-talk happens — getting flagged as
"applying everywhere indiscriminately" damages future credibility.

The list above is curated for genuine fit. Skip the longshots unless
specifically encouraged.
